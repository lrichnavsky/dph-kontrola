Microsoft 365 — přihlášení
🔒 Bezpečná architektura: Aplikace přistupuje k SharePointu přímo přes Microsoft Graph API pomocí OAuth2 PKCE flow. Data (dokladová čísla, základy DPH, DIČ) nikdy neopouštějí M365 tenant — neprochází žádným externím serverem. Pouze XML výstup (jen číselné součty) se generuje lokálně v browseru.
Aplikace musí být registrovaná v Azure AD s oprávněním Files.Read.All a Sites.Read.All
Registrace Azure App (jednorázové nastavení):

1. Otevři portal.azure.com → App registrations
2. New registration → název "DPH Kontrola" → Account type: "Single tenant"
3. Redirect URI: Web →
4. Po vytvoření zkopíruj Application (client) ID a vlož sem
5. API permissions → Add → Microsoft Graph → Delegated:
   • Files.Read.All
   • Sites.Read.All
6. Grant admin consent
